Welcome to STORE!

STORE is a complete example of CRUD based on a clothing and footwear e-commerce.

How has it been developed?

HTML

The language used to describe and define the content.

Bootstrap & Sass

To make all the visual part in a simple and orderly way.

Angular

To use components that represent the pages of the store, as well as to connect with the backend receiving the necessary data through the services.

Java Spring

Used to manage the API REST server that will work with the logic of the project, executing validations and handling CRUD actions.
